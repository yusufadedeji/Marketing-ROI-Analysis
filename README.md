# Marketing-ROI-Analysis
Simple Linear Regression of Marketing Data
This project analyzes marketing data to determine which advertising channel has the strongest impact on sales using Simple Linear Regression.

The analysis was performed using Python, pandas, seaborn, matplotlib, and statsmodels.

---

## Objectives

* Load and clean the dataset
* Perform exploratory data analysis
* Build a regression model
* Evaluate R-squared and p-values
* Interpret regression results
* Recommend the best marketing channel

---

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Statsmodels

---

## Files Included

* ROI Analysis.pdf
* Marketing ROI (2).csv
* README.md
* Marketing ROI Analysis.ipynb

---

## Results

The regression analysis showed a very strong relationship between the selected advertising channel and Sales.

The model achieved a high R-squared value, indicating strong predictive performance.
